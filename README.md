# Advent of Code 2023 in Clojure

Developed using JVM 21 and Clojure 1.11.

Run tests using `clj -X:test`.

```shell
$ clj -M -m aoc23.solver
Usage: clj -M -m aoc23.solver <day> <part> <puzzle-input-path>
Example: clj -M -m aoc23.solver 1 2 resources/input_01.txt
```

You have to supply the puzzle input files yourself.
